# 1091. 카드 섞기

<h2>느낀점</h2>
저번 주에 문제들이 많이 어려웠어서 이번 주도 쫄아있었는데, 첫 번째 문제는 다행히 무난하게 넘어간 것 같다. 처음에 메모리 초과가 떠서 뭐지 싶었는데, 카드를 섞는다는 의미를 잘못 파악했었다.
<hr>

<h2>풀이</h2>
<ol type="I">
    <li> 카드를 섞는 기능부터 구현하였다. 배열 2개를 만들고 S의 순서에 따라 임시 배열에 저장 후 다시 옮겨준다. </li>
    <li> 한 번 섞은 후 종료 조건을 파악한다(0,1,2,0,1,2 ...) </li>
    <li> 위의 과정을 반복한다 </li>
</ol>
<hr>

<h2>실수</h2>
문제를 잘 파악하자.
