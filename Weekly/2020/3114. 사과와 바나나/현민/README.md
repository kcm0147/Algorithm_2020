# 풀이

## 입력
1) row size
2) col size
3) board상태 : A11 은 사과나무가 11개, B3은 버네나 나무가 3개

## 조건
1) 불도저는 (0,0)에서 출발해서 (rowsize-1, colsize-1)에 도착!
2) 불도저는 아래, 아래-오른쪽, 오른쪽으로 1칸 움직일 수 있음.
3) 불도저가 지나가면 나무 없어짐

## 출력
1) 불도저가 땅을 나눴을 때, 아래 구역에 있는 사과나무 합 + 위 구역에 있는 바나나 나무 합

## 풀이법
1) 처음에 보고 완전탐색으로는 지수시간이 나오기 때문에 dp를 활용해야 겠다고 생각했다.
2) dp(x,y) : x,y에 불도저가 있을 때 최대 합
3) 총 시간 복잡도는 (문제의 수인 1500*1500) * (dp(x,y) 한번 실행 시간 1500) 으로 1500^3으로 풀었었다.
4) 하지만 시간내 될 줄알았는데 계산해보니 훨씬 넘었다.
5) 따라서 생각하다 부분합을 사용하면 될것 같았다.
6) 최종적으로 (문제의 수인 1500 * 1500) * (dp(x,y) 한번 실행 시간 : 상수 ) 으로 1500^2으로 충분히 풀 수 있었다.

## 느낀 점
1) 처음에 보고 쉽길래 골드 문제가 아닌줄 알았다.
2) 복병이 있었다! 다행히 부분합을 잘 떠올려 문제를 풀 수 있었다.
