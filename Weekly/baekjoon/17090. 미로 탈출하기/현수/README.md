# 17090. 미로 탈출하기

<h2>느낀점</h2>
    단순한 BFS 문제라고 생각했다. 풀면서 굳이 BFS를 안쓰고 DFS로 해결가능하다고 판단해서 바로 구현하였다. 배열 크기를 보고 시간초과가 안날 것이라 생각했는데 시간초과 뜨길래 성공 경로를 2, 실패 경로를 1, 미탐색 경로를 0으로 저장하였다.
<hr>
<h2>풀이</h2>
<ol type="I">
    <li> 문제의 형식대로 위치를 이동. </li>
    <li> 장외로 벗어나면 성공, 1 리턴하고 성공 경로를 재귀탈출하면서 성공경로를 뜻하는 2로 저장. 이미 지나갔던 경로에 도착하면 0 리턴. </li>
    <li> 모든 좌표에서 시작해보며 경로에 저장된 값에 따라 분기 </li>
</ol>
<hr>
<h2> 실수 </h2>
 500x500 배열이라서 시간초과가 날 것이라고 생각하지 못했던 점. -> 재귀로 캐싱없이 모두 탐색하면 시간초과가 뜰 가능성이 있다는 것을 알게 되었다.
