# 15999. 뒤집기

<h2>느낀점</h2>
>문제가 이해가 안되서 한참을 멍때리다가 접근방법을 떠올렸다.. 파이프 옮기기보다 오래 걸렸음
<hr style="color:green">

<h2>풀이</h2>
<ol type="I">
    <li>현재 상태까지 오기 위해서 이전에 어떤 상태에 있어야할지를 생각해보았다</li>
    <li>현재 서로 같은 색으로 둘러싸여 있는 격자가 바로 이전상태에서 반전된 색이어야한다.</li>
    <li>즉 뒤집었을 때 주변에 영향을 주지 않는 격자를 찾아내어 뒤집을지 말지에 대한 경우의 수를 계산한다.</li>
</ol>
