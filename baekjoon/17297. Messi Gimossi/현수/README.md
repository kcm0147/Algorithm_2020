# 17297. Messi Gimossi
<h2>느낀점</h2>
피보나치와 DP의 업그레이드 버전.. 메시 기모찌!
    
<hr>
<h2>풀이</h2>
<ol type="I">
    <li>  Messi 와 Messi Gimossi 를 기본 패턴으로 설정하고 각 글자수를 저장한다. </li>
    <li> 저장된 글자수를 가지고 피보나치 수를 만드는데, 공백의 개수도 포함해준다. </li>
    <li> 최대 수의 범위까지 만들어준 후, n의 수에서 마지막 수부터 차감하면서 기본 패턴의 글자수로 커버할 수 있을 때 까지 반복한다. </li>
    <li> 기본 패턴의 글자수보다 작아지면 해당 인덱스의 글자를 출력한다. <li>
</ol>
