# 1725. 히스토그램

<h2>느낀점</h2>
예전에 최대 부분합 구하기 문제랑 비슷하고, 알고2 과제로 했던 문제와 똑같은 문제여서 쉽게 풀었따! 분할정복문제를 조금만 더 풀어보면 감을 잡을 수 있을듯 하다
    
<hr>
<h2>풀이</h2>
<ol type="I">
    <li> 최대값이 나올 수 있는 범위는 중간을 기준으로 왼쪽, 오른쪽, 그리고 중간지점을 포함하는 영역이 존재한다. </li>
    <li> 양쪽으로 분할하여 최댓값을 구하고, 중간지점을 포함하는 영역에서 그리디 방법으로 값을 계산하여 최대값을 갱신해준다. </li>
</ol>
