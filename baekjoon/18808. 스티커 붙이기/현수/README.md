# 18808. 스티커 붙이기

<h2>느낀점</h2>
백준 문제 검색에 18808과 스티커 붙이기 로 검색하면 자꾸 안떠서 당황스러웠다! 문제 난이도는 골드3이라고 평가된거에 비해 쉬웠던거같다. 문제에서 노린 풀이법을 생각해보면서 선형대수의 회전변환 행렬도 떠올렸다. 하지만 조금 생각해보다가 너무 나간거 같아서 접고, 단순하게 좌표들이 변화하는 규칙을 찾아내었다. 이후 구현은 최대한 효율적으로 해보려고 하였지만, 조금 더 리팩토링이 가능할 것 같다.
<hr>

<h2>풀이</h2>
<ol type="I">
    <li> 한 점을 기준으로 각 스티커 내에 다른 지점의 상대적인 좌표들을 저장하여 둔다. </li>
    <li> 90도 회전 시 상대적인 좌표값의 변화를 적용한다. </li>
    <li> 이후 붙여지는 공간이 유효한 공간인지 판단 후, 유효하면 붙이고 해당 스티커가 차지하는 칸의 수를 최종값에 더한다. </li>
    <li> 문제의 조건대로 모든 지점에서 현재 모습의 스티커를 대어보고, 가능한 공간이 없다면 시계방향으로 90도 회전 후 다시 대어보는 과정을 반복한다. </li>
</ol>
