# 4056. 스-스-스도쿠
<h2>느낀점</h2>
스-스-스도쿠..! 모든 조건을 다 따졌다고 생각하고 제출하는데 50%에서 계속 틀렸다고 떠서 다른 스도쿠 문제에 같은 로직으로 제출했는데 정답이 떴다. 왤까..
    
<hr>
<h2>풀이</h2>
<ol type="I">
    <li> 문자열을 9줄 입력받는데, 한 줄 받을 때 마다 0을 가지는 좌표를 저장한다. </li>
    <li> 이후 각 사각형에 대해 0~8번 까지의 구역으로 분할한 후 각 구역에 포함된 숫자에 해당하는 인덱스는 true로 만들어준다. </li>
    <li> 이후 초기에 주어진 조건에 대해 더 진행할 수 있는지 가로/세로/구역별로 확인해준다. </li>
    <li> 진행이 가능하다면 0값의 좌표벡터 중 0번째 인덱스부터 숫자를 채워넣는다 <li>
    <li> 구역 내에서 가능한 숫자들의 벡터를 구해서 가로/세로 열과 중복되지 않으면 채워넣는다 <li>
</ol>
