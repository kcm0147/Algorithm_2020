# 2488. 별찍기 11

<h2>느낀점</h2>
보다가 보다가 규칙을 찾아낼 수가 없어서 인터넷에 검색해서 찾아봤다. 분할 정복을 사용하는 방법과 패턴에 대해 메모리를 복사하는 방법을 발견했다.
분할 정복에 대해 아직까지도 감이 잘 안온다.. 나누는 것 까지는 알겠는데 나눠지고 나서 돌아가는 방식에 대한 과정이 와닿지 않는다!! 분할정복 문제를 못풀었다고 생각하면 그러려니 할 수 있을텐데, 별찍기를 하지 못했다고 생각하니 우울해진다..ㅠ 
    
<hr>
<h2>풀이</h2>
<ol type="I">
    <li> 반복되는 최소 단위를 찾는다. (3*5 삼각형) </li>
    <li> 해당 패턴이 어떤 식으로 반복되는지 Bounding Box를 구상하여 한 좌표를 기준으로 파악한다. </li>
    <li> 배열에 메모리 복사를 이용하여 해당 패턴을 구현할 수 있다. </li>
</ol>
<hr>
<h2> 실수 </h2>
 dp에서 점화식 찾는거처럼 반복되는 규칙을 찾는 것이 아직 취약한 것 같다.
