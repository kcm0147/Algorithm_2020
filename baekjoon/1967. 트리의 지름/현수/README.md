# 1967. 트리의 지름

**느낀점**
> 나는 트리 문제를 보면 단말 노드에서 접근하려고 하는데 이 문제는 그렇게 해서 풀렸지만 다른 문제들도 그렇게 풀릴 지 잘 모르겠다. 적당히 생각하는 시간을 들여 무난하게 풀었다. <br>
