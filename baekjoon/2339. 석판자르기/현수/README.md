# 2339. 석판 자르기

<h2>느낀점</h2>
분할정복 말고 백트래킹과 부분합으로 구하였다가 크게 낭패를 보았다. 일단 2차원 부분합 구하는 것에서 헷갈려서 고민을 많이 했다. 결과적으로 분할정복만 가능한 문제였다.. 나눠진 부분들에 이전 방향과 다르게 잘라야 했던 것! 문제 파악이 정말 중요한거 같다..
<hr>

<h2>풀이</h2>
<ol type="I">
    <li> 분할해서 푸는 함수에 인자값으로 각 x,y 의 경계값을 넘겨준다 </li>
    <li> 해당 범위 내에 보석과 불순물의 개수를 카운트한다. </li>
    <li> 더 이상 진행이 불가능한 조건 3가지 </li>
    <li> 1. 자른 범위에 보석이 없는 경우 </li>
    <li> 2. 보석이 1개 남았지만 불순물도 1개 남은 경우 </li>
    <li> 3. 보석이 1개보다 많이 남았지만 불순물이 없어 더 자를 수 없는 경우 </li>
    <li> 해당 조건에서 더 이상 진행되지 않도록 0을 반환하고, 가능한 경우 계속 잘라나간다 </li>
</ol>
<hr>

<h2>실수</h2>
계산 과정을 처음부터 효율적으로 짜려다가 오히려 풀지도 못하고 복잡해지는 원인이 되었다. 최적화는 문제를 다 풀고 나서 시행하도록 해야겠다.. 분할정복 문제가 아직은 익숙하지 않아 혼동되었다. 조금 더 찾아봐야겠다.
